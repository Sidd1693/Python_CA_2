To run the application:-
1. Download JDK 8 and above.
2. Set Environment variable for java
3. Download "Application\lib" and "Application\Online_Food_Deivery.jar"
4. Open cmd in the folder where "Application\lib" and "Application\Online_Food_Deivery.jar" is placed.
5. type "Online_Food_Deivery.jar" and press enter
